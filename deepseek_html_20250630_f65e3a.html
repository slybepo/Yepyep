Unity Setup Guide

1

Download the Package

Download the SecureGuard Unity package from the download section below.

2

Import into Unity

In Unity, go to Assets > Import Package > Custom Package and select the downloaded file.

3

Add to Your Scene

Drag the SecureGuard prefab from the SecureGuard/Prefabs folder into your first scene.

4

Configure Settings

Select the SecureGuard object in your hierarchy and configure the settings in the Inspector:

// Example configuration public string gameId = "YOUR_GAME_ID"; public bool enableMemoryChecks = true; public float checkInterval = 5.0f;
5

Initialize in Code

Initialize SecureGuard in your game manager script:

using SecureGuard; void Start() { SecureGuardManager.Initialize(); SecureGuardManager.OnCheatDetected += HandleCheatDetection; } void HandleCheatDetection(string cheatType) { // Handle the cheat detection (disconnect player, show message, etc.) }
